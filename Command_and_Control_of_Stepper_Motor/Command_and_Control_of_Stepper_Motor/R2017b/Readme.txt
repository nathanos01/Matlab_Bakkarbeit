1) To setup the project in MATLAB, right click on "Init_project.m" and select "Run"
2) "Control_Panel.mlapp" is the main file of the project.
   Right click on it and select "Run" to start and run the App, or select "Open" to edit it
3) To close the project, right click on "Close_project.m" and select "Run"
4) "Control_Panel.prj" is the MATLAB Compiler project to build a standalone App with the MATLAB Compiler Toolbox
